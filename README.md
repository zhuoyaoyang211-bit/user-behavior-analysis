# 淘宝用户行为数据分析

基于淘宝用户全链路行为数据（浏览→收藏→加购→购买），完成从数据清洗、特征工程到模型建模的全流程数据分析项目。

## 一、项目背景

在电商行业精细化运营的发展趋势下，海量用户行为数据是挖掘用户偏好、优化商品运营、提升转化效率的核心资产。淘宝作为国内头部电商平台，拥有亿级用户与商品规模，用户从浏览、收藏、加购到购买的全链路行为数据中，蕴含着用户消费决策规律、商品价值潜力与平台转化瓶颈。

本项目以淘宝用户行为数据为载体，以"业务问题牵引 + 全流程实战"为核心模式，完整走完从数据处理、特征挖掘、模型搭建到业务落地的数据分析全链路。

### 五大核心业务问题

1. 如何高效处理大规模用户行为数据，保障数据质量与处理效率？
2. 如何搭建多维度特征体系，为用户行为预测提供高质量输入？
3. 如何通过机器学习模型精准预测用户购买行为，支撑精准运营？
4. 如何利用深度学习捕捉用户行为序列规律，进一步提升预测效果？
5. 如何将数据分析与建模结果转化为可落地的业务策略，创造实际业务价值？

## 二、项目目标

- **业务目标**：基于淘宝用户全链路行为数据，完成用户购买行为预测建模，输出用户分群、转化漏斗、商品价值等维度的深度洞察，提出可落地的用户运营与商品优化建议。
- **能力目标**：掌握大规模数据工程化处理、体系化特征工程、传统ML与深度学习建模全流程、业务导向的数据分析思维。

## 三、数据集说明

| 项目 | 说明 |
|---|---|
| 数据来源 | 淘宝用户行为公开数据集（1万用户采样版） |
| 记录数 | 12,256,906 行 |
| 独立用户 | 10,000 |
| 独立商品 | 2,876,947 |
| 独立类目 | 8,916 |
| 时间范围 | 2025-11-18 ~ 2025-12-18（1个月） |

### 字段说明

| 字段名 | 类型 | 说明 |
|---|---|---|
| user_id | string | 用户唯一标识 |
| item_id | string | 商品唯一标识 |
| item_category | string | 商品类目标识 |
| behavior_type | int | 行为类型：1=浏览(pv)、2=收藏(fav)、3=加购(cart)、4=购买(buy) |
| time | datetime | 行为发生时间 |
